# Модель для поддержания активности постоянных клиентов интернет-магазина «В один клик»

## Постановка задачи

Разработать решение, которое позволит персонализировать предложения постоянным клиентам, чтобы увеличить их покупательскую активность. Целевая переменная — **покупательская активность** (снизилась / не снизилась).

**Данные:** информация о поведении покупателей на сайте, коммуникациях с покупателем, продуктовом поведении, выручке и времени, проведённом на сайте.

## Ход исследования

1. **Загрузка и анализ данных** — изучение общей информации из датасетов
2. **Предобработка данных** — обработка дубликатов, исправление опечаток, приведение типов данных
3. **Исследовательский анализ данных (EDA)** — отбор клиентов с активностью не менее трёх месяцев, визуализация признаков
4. **Объединение таблиц** — слияние `market\_file`, `market\_money`, `market\_time` и `money`
5. **Корреляционный анализ** — анализ признаков в итоговой таблице
6. **Обучение моделей (пайплайны)** — перебор моделей с подбором гиперпараметров
7. **Анализ важности признаков** — с помощью SHAP-значений
8. **Сегментация покупателей** — формулировка рекомендаций для бизнеса

## Используемые модели

|Модель|Описание|
|-|-|
|`LogisticRegression`|Логистическая регрессия (L1/L2-регуляризация)|
|`DecisionTreeClassifier`|Дерево решений|
|`KNeighborsClassifier`|Метод k-ближайших соседей|
|`SVC`|Метод опорных векторов|

Подбор гиперпараметров выполнялся через `RandomizedSearchCV` с кросс-валидацией (cv=5). Метрика качества — **ROC AUC**.

## Результат

**Лучшая модель — Логистическая регрессия** с параметрами:

* `C = 4` (умеренная регуляризация)
* `penalty = 'l1'` (L1-регуляризация, отбор значимых признаков)
* `solver = 'liblinear'`
* **ROC AUC = 0.92**

### Ключевые признаки (по SHAP-анализу)

* Доля акционных покупок
* Среднее количество просмотренных категорий за визит
* Время, проведённое на сайте в предыдущем и текущем месяцах
* Количество неоплаченных товаров в корзине за квартал

### Выводы и рекомендации

Выявлен сегмент клиентов с высокой вероятностью снижения активности — пользователи, часто покупающие по акции. Для удержания этого сегмента рекомендуется:

* Увеличить количество рассылок со скидками
* Обновлять информацию о скидках на главной странице
* Акцентировать внимание на скидках в категориях: товары для детей, домашний текстиль, косметика
